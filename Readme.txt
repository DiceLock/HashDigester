This file contains a summary of what you will find in each of the folders that
make up your HashDigester SDK library Visual C++ Project.

Load with Visual Studio C++ 2008 or compatible HashDigester.sln file and will
be able to generate and modify HashDigester DLL file.

/////////////////////////////////////////////////////////////////////////////


folder: \HashDigester 

------
   This is the folder that contains all HashDigester SDK library files.
   It contains:
   - HashDigester.sln (Microsoft Visual Studio Solution),
   - HashDigester.ncb (Visual C++ Intellisense Database),
   - Readme.txt       (this file),

folder: \HashDigester\HashDigester
------
   This is the folder that contains all HashDigester .h and .cpp files.
   The main header file is "hashDigester.h", it contains the references to all
   other header files. Each random test has it's own header file allowing to 
   insert individually and specifically hash algorithms in your applications.


folder: \HashDigester\HashDigester\Release
------
   This is the folder that contains generated HashDigester files
   with Visual Studio C++ 2008.

folder: \HashDigester\Release
------
   This is the folder that contains all generated and final products:
   - HashDigester.dll (DLL file implementing random number tests),
   - HashDigester.exp (Exports Library File),
   - HashDigester.lib (Object File Library),
   - HashDigester.pdb (Program Debug Database),
